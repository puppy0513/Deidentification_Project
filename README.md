## ✨ 해커톤 소개 
- 주어진 데이터 셋 중 하나를 선택하여 비식별화 처리한다. 
- K-익명성, L-다양성을 필수적으로 설정해야 한다. 
- 비식별화 처리된 데이터를 이용하여 분석모델을 만들것.
- 데이터분석을 통해 스토리를 만들어낼것.
- 비식별화처리된 데이터와 Raw Data의 결과가 유사할것.

## 👨‍👦 Member
- 배준현 : 데이터 분석, 스토리 작성
- 차광호 : 가명처리
